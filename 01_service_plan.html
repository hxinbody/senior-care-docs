AI 전화통화 기반 시니어 헬스케어 서비스 기획안

일본 50~70대 시니어를 타겟으로, InBody 체성분 데이터와 AI 전화통화 기술을 결합한 헬스케어 서비스
2026.04.27 | FitPlus DataScience Team
CONTEXT

배경 및 목표

일본 50~70대 시니어를 타겟으로, InBody 체성분 데이터 + AI 전화통화 기술을 결합한 헬스케어 서비스를 기획한다. 정부 통계(国民健康・栄養調査, 中高年者縦断調査 등), JAGES 코호트, 민간 리서치를 근거로 삼고, FitPlus가 이미 보유한 데이터 파이프라인을 최대한 활용한다.

DATA

A

AI 전화통화로 할 수 있는 것

A-1

정기 건강 체크인

Scheduled Health Check-in
  • 주 1~2회 정기 전화: "이번 주 몸 상태는 어떠세요?" 식 대화
  • 주관적 건강 상태 수집: 통증 부위, 피로도, 식욕, 수면 질, 기분 (5점 척도 음성 입력) R1
  • 낙상 경험 확인: "지난 한 주간 넘어지거나 비틀거린 적 있으세요?" R2
  • 복약 확인: "혈압약/당뇨약 잘 드시고 계세요?" R3
  • 생활 활동 확인: 외출 빈도, 사회 활동 참여, 가사 활동 수행 여부 R4
A-2

InBody 측정 결과 음성 해설

Voice-Delivered Report
  • 측정 직후 자동 전화: InBody 측정 → 데이터 동기화 → AI가 전화로 결과 설명
  • FitScore 변화 알림: "지난달 대비 핏스코어가 15점 올랐습니다"
  • 트렌드 코멘트 음성 전달: 기존 TrendReport CoreSentence를 음성으로 변환 R5
  • 클러스터 변화 설명: "같은 연령대에서 상위 35%에서 15%로 올라가셨습니다" R6
  • 위험 지표 경고: VFA 상승, SMM 급감, WED 이상 등을 쉬운 말로 설명 R7
A-3

동기부여 및 행동 코칭

Motivational Coaching
  • 목표 설정 대화: "이번 달 목표를 정해볼까요? 하루 6,000보 걷기는 어떠세요?" R8
  • 달성 축하 전화: DayLog 기반 목표 달성 시 축하 + 다음 목표 제안 R9
  • 운동 리마인더: "오늘 날씨가 좋은데, 산책 나가실 계획 있으세요?"
  • 식사 가이드: "단백질 섭취가 부족해 보여요. 두부나 생선 한 끼 추가해보세요" R10
A-4

인지기능 간이 스크리닝

Cognitive Screening
  • 시간 지남력: "오늘이 무슨 요일인지 아세요?" R11
  • 단기 기억: "아까 제가 말씀드린 세 단어를 기억하세요?" R12
  • 언어 유창성: "1분 동안 동물 이름을 최대한 많이 말씀해주세요" R13
  • 대화 패턴 분석: 응답 시간, 단어 다양성, 문장 완성도의 시계열 변화 추적 R14
  • 우울 경향 감지: 음성 톤, 대화 참여도, 부정적 표현 빈도 모니터링 R15
A-5

프레일티(허약) 조기 감지

Frailty Early Detection
  • J-CHS 기준 변형 5문항: R16
  • "체중이 줄었다고 느끼세요?" (체중 감소)
  • "계단을 오를 때 예전보다 힘드세요?" (근력 저하)
  • "쉽게 피곤해지세요?" (피로)
  • "최근 걷는 속도가 느려진 것 같으세요?" (보행속도)
  • "밖에 나가는 횟수가 줄었나요?" (신체활동)
InBody SMM 감소 추세 + 주관적 근력 저하 보고 → 근감소증 교차 검증 R17 R18
A-6

긴급 상황 대응

Emergency Response
  • 이상 징후 에스컬레이션: 전화 응답 없음 (3회 연속), 혼란스러운 대화, 낙상 보고 R19
  • 자동 알림: 보호자/의료기관에 즉시 통보 R20
  • 긴급 연락망 연동: 등록된 가족/주치의에게 SMS/전화 전달
B

InBody + FitPlus 데이터로 할 수 있는 것

B-1

근감소증(Sarcopenia) 모니터링

  • 기존 SMM/FFMI 시계열 분석 활용 R21
  • AWGS 2019 기준 적용: SMI = ASM/height² < 7.0 kg/m² (남), < 5.7 kg/m² (여) R22
  • 분절 체성분(BCofPart): 상하지 근육량 비대칭 감지 → 편측 약화 경고
  • ASM 산출: Measure_BCofPart 테이블의 사지 근육량 활용
B-2

내장지방 위험도 추적

  • VFA/VFL 시계열 트렌드: 내장지방 면적 100cm² 초과 시 경고 R7
  • WHR 연동: 복부비만 지표와 내장지방의 교차 분석 R23
  • 대사증후군 위험 스코어: BMI + VFA + WHR 종합 판정 R24
B-3

체수분/부종 모니터링

  • WED 추이 분석: 심부전/신장질환 조기 징후 R25
  • Phase Angle 백분위: 기존 BWA 엔드포인트 활용 → 영양 상태/세포 건강 지표 R26
  • 부종 레벨 분류: 기존 wed-level API 직접 활용 가능
B-4

FitScore 시니어 특화 버전

  • 60세 이상 확장 모델: 현재 20~60세 → 60~80세 구간 추가 R27
  • 근감소증 가중치 추가: SMM/FFMI 변화를 FitScore에 반영 R21
  • JP 역학 데이터 기반: EPOCH-JAPAN, Matsuo et al. 근거로 70대 BMI_STD 설정 R28 R29
  • 프레일티 통합 점수: FitScore + 프레일티 체크 → 종합 건강 등급 R30
B-5

인구 집단 비교 리포트

  • BigData_HistogramInfo 활용: 동일 연령대/성별 내 위치 시각화
  • Cluster Index 시니어 특화: 50~70대 전용 9x9 그리드 재보정
  • 연간 Recap 확장: 시니어 전용 항목(근감소증 추이, 프레일티 경향) 추가
B-6

InBody Everyday(가정용) 연동

  • StackHistory 테이블 활용: 매일 가정에서 측정 → 고빈도 데이터
  • 이상치 즉시 알림: 하루 단위 체중/체수분 급변 감지 → AI 전화로 확인
C

현재 역량 밖이지만 있으면 유용한 것

C-1

웨어러블 바이탈 연동

  • 연속 심박수/HRV: 자율신경 기능, 스트레스 수준
  • 혈압 자동 기록: 고혈압 관리 (일본 시니어 60%+ 고혈압) R3
  • SpO2 산소포화도: 수면무호흡, 호흡기 문제 조기 감지
  • 체온 추이: 감염/염증 조기 경고
DayLog IsHealthConnect 필드 이미 존재 → Apple HealthKit / Google Fit 확장 가능
C-2

혈액검사/의료 데이터 연동

  • HbA1c, 공복혈당: 당뇨 관리 (유병률 ~20%) R3
  • LDL/HDL 콜레스테롤: 심혈관 위험도
  • eGFR/크레아티닌: 신장 기능 (WED 데이터와 교차)
  • 알부민/프리알부민: 영양 상태 (저영양 = 프레일티 핵심 원인)
特定健診 데이터 API 또는 PHR 연동
C-3

GPS/위치 기반 활동 모니터링

  • 외출 빈도/반경 자동 추적: 사회적 고립 조기 감지
  • 방문 패턴 분석: 병원/운동시설 방문 생활 활동 수준 평가
  • 낙상 감지 자동 알림: 스마트폰 가속도센서 + 위치 정보
C-4

음성 바이오마커 분석

  • 성대 떨림/음성 주파수 분석: 파킨슨병, 우울증 조기 징후 R15
  • 발화 속도/멈춤 패턴: 인지기능 저하 장기 추적 R14
  • 호흡 패턴 분석: 호흡기 건강 모니터링
AI 전화 음성 데이터를 2차 분석하는 파이프라인
C-5

식사 영양 자동 분석

  • 사진 기반 식사 기록: AI 이미지 분석으로 칼로리/영양소 추정
  • 단백질 섭취량 추적: 근감소증 예방 핵심 (1.0~1.2g/kg/day 권장) R10
  • 수분 섭취 모니터링: 탈수 예방 (시니어 갈증 감각 저하)
C-6

의료기관/개호시설 연동 플랫폼

  • 주치의 대시보드: 환자별 체성분 트렌드 + AI 전화 요약 열람
  • 개호매니저 리포트: 요개호 인정 판정용 객관적 데이터 제공
  • 약국 연동: 복약 이력 + AI 전화 복약 확인 결과 대조
  • LIFE 연동: 후생노동성 科学的介護情報システム 연계 R31
C-7

사회적 고립 방지 기능

  • AI 말동무: 건강 체크 외 일상 대화, 추억 이야기, 뉴스 토론
  • 지역 커뮤니티 연결: 근처 운동 교실, 건강 이벤트 정보 안내
  • 가족 공유 리포트: 원거리 가족에게 주간 건강 요약 자동 전송
C-8

보험/건강경영 연동

  • 特定健診 대응: 메타볼릭 신드롬 판정과 FitScore 연계
  • 건강경영 기업 대응: 정년 후 고령 재고용자 건강관리 패키지
  • 보험사 연동: 건강 개선 실적 기반 보험료 할인 데이터 제공

핵심 차별점: AI 전화는 스마트폰 앱이 필요 없습니다. 디지털 소외 계층인 시니어가 익숙한 전화 한 통으로 InBody 체성분 데이터 기반의 전문적인 건강 관리를 받을 수 있습니다. 이것이 기존 헬스케어 앱과의 가장 큰 차이점입니다.
